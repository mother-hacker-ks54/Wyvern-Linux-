What is Wyvern

is a Linux distribution based on Ubuntu designed to solve problems in the field of cybersecurity and consisting mainly of free and open source software. Wyvern Linux use default desktop has been Budgie since version 22.04.

Open-source project

You can test secure your open source apps, patch the full stack, from kernel to library and other applications.

Lightweight and not rigid distribution

Light redesigned graphical environment, only profile applications, handy scripts and much more.

Smart and very comfortable for tasks

a modern graphical environment makes the system flexible and aesthetically pleasing. The system is built in such a way that it is not a problem to customize it for yourself.

Aimed at the field of information security

System optimized to reduce the amount of work, so a professional can just sit down and go. Task on web, crypto and more.

Modern UNIX system

The distribution kit corresponds to the trends of modern systems in order to make it pleasant for you to work with it.

Semi-rolling release

Semi distribution support to create better versions and add new utilities and software.
